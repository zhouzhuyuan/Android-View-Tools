## 项目简介

AndroidTools 是一个包含多种实用自定义控件和工具类的 Android 工具仓库，方便开发者快速集成常用功能。

## 快速开始

1. 克隆本项目到本地：
   ```sh
   git clone git@github.com:zhouzhuyuan/Android-View-Tools.git
   ```
2. 在 Android Studio 中打开项目。
3. 按需在 app 或其他模块中引用 atvs 里的自定义控件。
